# 자동차 경주 게임

## 👀 기능 목록

- InputReader
    - [x] 경주할 자동차 이름
    - [x] 시도할 횟수
- Validator
    - [x] 자동차 이름은 `,` 기준으로 5자 이하
    - [x] 시도할 횟수로 입력 받은 값이 숫자인지 검증
    - [x] 사용자가 유효하지 않은 값을 입력하면 `IllegalArgumentException`를 던지고, 에러 메시지 출력 후 다시 입력 받기
- MessagePrinter
    - [x] 자동차 이름을 입력하라는 메시지
    - [x] 시도할 횟수를 묻는 메시지
    - [x] 다음에 출력되는 메시지가 실행 결과임을 알리는 메시지를 출력
    - [x] 실행 결과(게임 진행 상황) 출력
    - [x] 최종 우승자 출력
- Game
    - [x] `Car`배열과 반복 횟수를 초기화
    - [x] 턴을 진행시킴
    - [x] 반복하는 횟수만큼 게임을 반복하기
    - [x] 가장 멀리간 자동차의 위치 계산
    - [x] 최종 우승자를 계산
- Car
    - [x] 자동차를 전진시키기
- Utility
    - [x] 0 ~ 9 사이에서 무작위 값을 생성
